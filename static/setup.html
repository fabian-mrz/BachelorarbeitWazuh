Setup Instructions for Wazuh

  1. Step 1: Configure Certificates

    Generate and configure SSL certificates for the Wazuh Dashboard and service.

                        
                        # Follow the official documentation to generate and configure SSL certificates
                        https://documentation.wazuh.com/current/user-manual/wazuh-server-cluster.html#certificates-deployment

  2. Step 2: Randomize JWT

    Update the JWT settings in the configuration file to enhance security.

                        
                        # Edit the config.json file to randomize JWT
                        vi config/config.json

  3. Step 3: Change API Tokens

    Update the API tokens in the configuration file.

                        
                        # Modify the API tokens in config/config.json
                        vi config/config.json
